
# Chien Robot

## Description
Projet de robot mobile avec contrôle de servomoteurs et encodeurs.

## Matériel utilisé
- Arduino
- Servomoteurs (MG90S / FS90R)
- Encodeurs
- Module moteur

## Fonctionnalités
- Déplacement du robot
- Mouvement de la queue
- Contrôle des moteurs

## Utilisation
1. Ouvrir le fichier .ino dans Arduino IDE
2. Téléverser sur la carte
